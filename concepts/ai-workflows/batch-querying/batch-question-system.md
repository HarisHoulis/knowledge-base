# Improving /grill-me with Batch-Based Question System

Matt Pocock explores a new approach for the /grill-me skill by switching from a one-by-one question format to a batch-based system. Instead of asking questions sequentially, the skill now groups questions into rounds, allowing multiple questions to be processed together. This change reduces wait times between questions and minimizes context switching, making the interaction smoother and more efficient. The batch system also handles dependencies between questions, ensuring that the order of questions within a round respects logical constraints. Overall, this improvement enhances the user experience by streamlining the questioning process and reducing cognitive load.

- Switched from one-by-one questions to batch-based rounds.
- Reduces wait times by processing multiple questions at once.
- Minimizes context switching for the user.
- Handles question dependencies within rounds.
- Overall improves efficiency and user experience.
