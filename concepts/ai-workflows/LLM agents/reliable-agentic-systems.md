# Building Reliable Agentic AI Systems

This article describes a project by Martin Fowler's colleagues in collaboration with Bayer, where they built an intelligent research assistant using LLMs. The system enables pharmaceutical researchers to query decades of information buried in PDF reports, evolving from simple keyword-based search to a sophisticated assistant capable of answering complex questions and drafting regulatory documents. The project highlights the challenges and approaches to making LLM-based systems reliable and useful in high-stakes domains like pharmaceutical research.

- The system evolved from keyword search to an intelligent research assistant using LLMs.
- It allows pharmaceutical researchers to query decades of information from PDF reports.
- The assistant can answer complex questions and draft regulatory documents.
- The project demonstrates how to build reliable LLM-based systems for high-stakes domains.
