# Now in Android #90

This edition of Now in Android covers several updates for Android developers. A new visual identity for the Android brand was announced, featuring an updated logo and a 3D version of the Bugdroid mascot, reflecting Android's ethos of being open and inclusive. ART (Android Runtime) is now updatable via Google Play system updates, allowing hundreds of millions of devices to receive performance improvements, security fixes, and new OpenJDK APIs. The upcoming ART 14 will include OpenJDK 17 support and compiler/runtime optimizations. Dagger and Hilt version 2.48 were released, including an alpha of their KSP processors, aiming to replace KAPT for faster builds. Jetpack Compose Accompanist libraries received an update, with some being deprecated. New guidance was published for modularization patterns, Java versions in Android builds, and Compose animations, including a quick-start guide and an improved decision tree. The Now in Android open-source app now uses screenshot tests to verify visual appearance of Compose components. (Source: Now in Android #90 article)

- Android brand gets a new logo and 3D Bugdroid as part of an updated visual identity.
- ART (Android Runtime) is being delivered via Google Play system updates, bringing ART 14 with OpenJDK 17 and performance gains to compatible devices.
- Dagger/Hilt 2.48 introduces alpha KSP processors, moving away from KAPT for faster annotation processing.
- Jetpack Compose Accompanist libraries are being deprecated or replaced; developers should check the latest guidance.
- New and updated developer guides: modularization patterns, Java version selection, and Compose animations (including a quick guide and decision tree).
